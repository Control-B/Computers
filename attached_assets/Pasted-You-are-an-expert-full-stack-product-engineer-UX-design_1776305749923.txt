====================================
AI UX REQUIREMENTS
====================================

Implement a hybrid AI + form experience.

Requirements:
- user can type a natural-language calculation request in the hero input
- the system should parse common phrases and auto-fill calculator fields where possible
- user can still manually edit the form
- calculation runs using deterministic code
- AI insight is generated from the calculation result object

For MVP:
- build a lightweight parser for common phrases
- no need for full LLM integration yet
- structure code so real AI API can be added later

Examples:
- “calculate compound interest on 10000 at 5% for 5 years”
- “what’s my mortgage payment on 300k with 20 percent down at 6.5 percent”
- “if I bought 2 bitcoin at 30000 and sold at 45000 what is my profit”

The parser should:
- detect which tool is intended
- prefill fields
- route or update the current tool page appropriately
